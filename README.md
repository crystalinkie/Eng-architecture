# 우리 아이 영어 성향 테스트
## 아이 맞춤 영어 설계소 | 4T™ 진단

### 배포 방법 (Vercel - 무료)

**1단계: GitHub에 올리기**
1. github.com 가입
2. 새 repository 만들기 (이름 예: `4t-english-test`)
3. 이 폴더 안의 파일들을 모두 업로드

**2단계: Vercel 배포**
1. vercel.com 가입 (GitHub 계정으로 로그인)
2. "Add New Project" 클릭
3. 방금 만든 GitHub repository 선택
4. "Deploy" 클릭

**3단계: 완료!**
- 자동으로 `your-project.vercel.app` 링크 생성
- 이 링크를 카카오톡, 네이버 카페에 공유하세요
